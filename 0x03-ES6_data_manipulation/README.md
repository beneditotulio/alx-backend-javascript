## Setup

### Install NodeJS
1. curl -sL https://deb.nodesource.com/setup_12.x -o nodesource_setup.sh
2. sudo bash nodesource_setup.sh
3. sudo apt install nodejs -y
4. nodejs -v
5. npm -v
